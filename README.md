# 🚀 Talio – Job Portal

Talio is a modern job portal web application built with **Next.js** that connects job seekers with employers. It allows users to explore job opportunities, apply for positions, and enables employers to post and manage job listings efficiently.

## 📌 Features

### 👨‍💼 Job Seeker
- Create account and login
- Browse available job listings
- Search and filter jobs
- Apply for jobs
- Track application status

### 🏢 Employer
- Employer registration & login
- Post new job openings
- Edit and delete job listings
- View applicants for jobs

### 🔐 Authentication
- Secure authentication system
- Role-based access control (Job Seeker / Employer)
- Session handling

---

## 🛠️ Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Frontend:** React.js
- **Styling:** Tailwind CSS
- **Backend:** Next.js API Routes
- **Database:** MySQL
